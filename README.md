These are my dotfiles. They are heavily inspired by the [Hashrocket
Dotmatrix][dotmatrix].

### Installation

```
$ mkdir ~/oss
$ cd ~/oss
$ git clone https://github.com/jwworth/dotfiles.git
```

Next, run the install script:

```
$ bin/install
```

To set up a new Mac, run:

```
$ ./.macos
```
